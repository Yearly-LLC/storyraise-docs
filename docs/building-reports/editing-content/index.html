Editing content

Everything in a Storyraise report is edited directly on the canvas — what you see while editing is what readers see. This page covers how sections, blocks, and elements fit together and how to edit each.

How content is organized

  • A section is one page of your report.
  • Inside a section are blocks — content areas with their own style and layout settings.
  • Inside blocks are elements — the individual pieces of content: text, images, videos, charts, and more.

Click anything on the canvas to select it; the settings panel shows the options for whatever is selected.

Editing text

Click into any text and type. You can adjust a text element's font, size, color, alignment, line height, and letter spacing from its settings. For report-wide typography, set your primary and secondary fonts in the brand kit instead of styling text element by element.

Elements you can add

  • Text — paragraphs and headings
  • List — bulleted and numbered lists
  • Button — calls to action linking anywhere
  • Image — photos and graphics, with sizing, alignment, corner rounding, and shadow options
  • Video — YouTube and Vimeo embeds
  • Audio — audio player
  • Counter — animated numbers, great for key stats
  • Chart — pie, bar, and line charts
  • Table — tabular data
  • Accordion — collapsible content for FAQs and details
  • Card — framed content container
  • Horizontal rule — a visual divider
  • Twitter / Facebook embeds — social posts inside your report
  • Embed — any other embeddable content via iframe
  • Layout helpers — Box, Row, and Mini Grid for arranging elements side by side
  • Repeater element — repeating items inside a Repeater block

Styling blocks

Select a block to open Set Block Style & Layout, where you control its background, spacing, and arrangement. Brand kit colors appear in every color picker, so staying on brand is one click.

Working safely

  • If a teammate is editing a section, it's locked until they're done — edit any other section in the meantime.
  • Deleted sections can be restored.
